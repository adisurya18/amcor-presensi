Sebelum mulai
Setup ini hanya dilakukan sekali. Setelah selesai, semua data member & check-in tersimpan terpusat di Google Sheets milikmu.
1
Buat Google Sheets baru

Buka sheets.google.com → buat spreadsheet baru → beri nama AMCOR Presensi.

2
Buka Apps Script

Di Google Sheets: menu Extensions → Apps Script → hapus semua kode yang ada → paste kode berikut:

Klik tombol di bawah untuk lihat script

3
Deploy sebagai Web App

Klik Deploy → New deployment → pilih type Web app → Execute as: Me → Who has access: Anyone → Klik Deploy → Authorize → Copy URL yang muncul.
